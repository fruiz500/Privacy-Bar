Page Privacy Help

How the Page Tab Works
The Page tab is the default view for most websites. It activates automatically if the page doesn't contain password fields, large text inputs, or encrypted blobs.
How to Use Encrypted Page Notes
You can store private notes for specific websites that only appear when you visit that page.
  1. Type your note into the text box.
  2. Ensure your Master Key (MK) is filled.
  3. Click Save.
  4. The next time you visit, the note decrypts automatically if your MK is filled.
  5. Read-Once: Click Read-Once to save a note that will be deleted from storage immediately after it is decrypted and displayed once.
  6. Delete: Click Delete and confirm in the popup to remove a saved note.
Note: Saving permanent notes requires a Pro license. Read-once notes can be saved in free mode.
Isolating Pages (Safe Space) PRO
To prevent other browser extensions from "seeing" what you do on a sensitive page:
  1. Click Send Page to Safe Space.
  2. A new tab opens containing the page inside an isolated frame. Other extensions cannot interact with this frame or log your keystrokes.
  3. Warning: Some high-security sites (like banks) may block this isolation and result in a blank page.
Note: This feature requires a Pro license.
Sanitizing Dangerous Pages PRO
If you suspect a page is malicious or contains tracking scripts:
  1. Click Open Sanitized Page.
  2. A new tab will open with a stripped-down version of the site. All executable code, images, and malicious metadata-stripping content are removed.
  3. Most links will still work, but subsequent pages you visit from there are not automatically sanitized.
Note: This feature requires a Pro license.
